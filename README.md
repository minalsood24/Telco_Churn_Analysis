# 📊 Telco Customer Churn Analysis

This project analyzes customer churn behavior in a telecom company using Python and data analysis techniques.

---

## 🎯 Objective
To identify key factors that influence customer churn and provide insights to improve customer retention.

---

## 📁 Dataset
The dataset includes:
- Customer demographics
- Tenure
- Monthly charges
- Contract type
- Payment methods
- Churn status

---

## 🔍 Key Steps
- Data Cleaning & Preprocessing  
- Handling Missing Values  
- Exploratory Data Analysis (EDA)  
- Data Visualization  
- Correlation Analysis  

---

## 📊 Key Insights
- Customers with month-to-month contracts have higher churn rates  
- Higher monthly charges increase churn probability  
- Long-term customers are less likely to churn  
- Electronic payment users show higher churn  

---

## 🛠 Tools Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook
