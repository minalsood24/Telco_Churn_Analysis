# 📊 Telco Customer Churn Analysis

This project focuses on analyzing customer churn in a telecom company using data analysis techniques in Python.

---

## 🎯 Objective
The goal of this project is to identify key factors that influence customer churn and provide insights to improve customer retention.

---

## 📁 Dataset
The dataset contains customer information such as:
- Gender
- Tenure
- Monthly Charges
- Contract Type
- Payment Method
- Churn Status

---

## 🔍 Key Steps Performed
- Data Cleaning & Preprocessing
- Handling Missing Values
- Exploratory Data Analysis (EDA)
- Data Visualization
- Correlation Analysis

---

## 📊 Key Insights
- Customers with month-to-month contracts have higher churn rates  
- Higher monthly charges are linked to increased churn  
- Long-term customers are less likely to churn  
- Electronic payment users show higher churn  

---

## 🛠 Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook

## 📌 Conclusion
This analysis helps businesses understand customer behavior and take proactive steps to reduce churn.
